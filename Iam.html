🔐 AWS IAM

🔐 Identity and Access Management (IAM)

IAM allows you to securely manage access to AWS services and resources.

📚 Lesson Information

⏱️ Estimated Time: 20 Minutes

🎯 Level: Beginner

📖 Exam Importance: Very High

What is IAM?

IAM is a global AWS service used to control who can access AWS resources and what actions they can perform.

Why IAM Matters

  • Control access to AWS resources
  • Improve security
  • Apply least privilege access
  • Manage users and permissions centrally

🔐 IAM Structure

AWS Account
     │
     ├── Users
     │
     ├── Groups
     │
     ├── Roles
     │
     └── Policies

🎯 Exam Tip

Users = People Groups = Collection of Users Roles = Temporary permissions Policies = Permission documents

📝 Quick Quiz

Which IAM component provides permissions? A. User B. Policy C. Group Answer: B
