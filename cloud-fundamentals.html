Cloud Fundamentals is the foundation of AWS and one of the first topics for the AWS Solutions Architect Associate certification.

What is Cloud Computing?

Cloud Computing is the delivery of computing services such as servers, storage, databases, networking and software over the internet.

Benefits of Cloud Computing

  • Pay only for what you use
  • Scalability
  • High Availability
  • Global Reach
  • Improved Security

AWS Global Infrastructure

AWS provides infrastructure across the world through Regions, Availability Zones and Edge Locations.

🌍 AWS Global Infrastructure

AWS Region
    │
    ├── Availability Zone A
    │
    ├── Availability Zone B
    │
    └── Availability Zone C

Region

A Region is a geographical area containing multiple Availability Zones. Example: eu-west-2 (London)

Availability Zone (AZ)

An Availability Zone is one or more data centers within a Region. Example: London Region → AZ-A, AZ-B, AZ-C

Edge Location

Edge Locations are used by CloudFront to cache content closer to users, reducing latency and improving performance.

Exam Tip

Regions contain multiple Availability Zones. CloudFront uses Edge Locations. This is a very common AWS exam question.
