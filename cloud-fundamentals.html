Cloud Fundamentals is the foundation of AWS and one of the first topics for the AWS Solutions Architect Associate certification.

What is Cloud Computing?

Cloud Computing is the delivery of computing services such as servers, storage, databases, networking and software over the internet.

Benefits of Cloud Computing

Cloud Service Models

Cloud services are commonly categorized into three models: Infrastructure as a Service (IaaS), Platform as a Service (PaaS), and Software as a Service (SaaS).

  • IaaS – Amazon EC2
  • PaaS – AWS Elastic Beanstalk
  • SaaS – Gmail, Microsoft 365
  • Pay only for what you use
  • Scalability
  • High Availability
  • Global Reach
  • Improved Security

Cloud Deployment Models

  • Public Cloud
  • Private Cloud
  • Hybrid Cloud

AWS Shared Responsibility Model

AWS follows a Shared Responsibility Model. AWS is responsible for security of the cloud, including physical infrastructure, networking and hardware. Customers are responsible for security in the cloud, including IAM permissions, operating systems, applications and data.

Real-World Use Cases

  • Netflix streaming services
  • Online banking applications
  • E-commerce platforms such as Amazon
  • Global content delivery networks

AWS Global Infrastructure

AWS provides infrastructure across the world through Regions, Availability Zones and Edge Locations.

Region

A Region is a geographical area containing multiple Availability Zones. Example: eu-west-2 (London)

Availability Zone (AZ)

An Availability Zone is one or more data centers within a Region. Example: London Region → AZ-A, AZ-B, AZ-C

Edge Location

Edge Locations are used by CloudFront to cache content closer to users, reducing latency and improving performance.

AWS Solutions Architect Exam Notes

  • Regions contain multiple Availability Zones.
  • Availability Zones consist of one or more data centers.
  • CloudFront uses Edge Locations for content caching.
  • AWS follows the Shared Responsibility Model.
  • Cloud computing provides scalability, elasticity and pay-as-you-go pricing.
